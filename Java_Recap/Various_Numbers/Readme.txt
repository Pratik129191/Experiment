#Run_Guide

1) Install Java JDK in Machine.
2) Open Command Prompt in that folder where this program is downloaded.
3) TYPE or COPY-PASTE the below command in CMD.
        javac Number_Checking.java
        java Number_Checking
4) It's Done.......Now Enjoy it........
